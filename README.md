# homebrew-liqoria

Homebrew tap for [Liqoria](https://www.liqoria.com) — Mac music player for Apple Music, Spotify, YouTube, and more.

## Install
```bash
brew tap aviorrok/liqoria
brew install --cask liqoria
```

## Update
```bash
brew upgrade --cask liqoria
```

## Uninstall
```bash
brew uninstall --cask liqoria
brew untap aviorrok/liqoria
```

## Requirements

- macOS 15.4 (Sequoia) or later
